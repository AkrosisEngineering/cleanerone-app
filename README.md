# CleanerOne Landing Page Starter

Bu klasor, baska bir repoya tasinabilecek sekilde hazirlanmis statik landing page paketidir.

## Icerik

- `index.html`: ana landing page
- `style.css`: tum stiller
- `app.js`: mobil menu ve reveal animasyonlari
- `privacy/index.html`: privacy policy sayfasi taslagi
- `terms/index.html`: terms sayfasi taslagi
- `assets/*.svg`: placeholder logo ve app screenshot gorselleri

## Ilk degistirilecek yerler

1. `index.html` icindeki App Store ve TestFlight linkleri
2. footer icindeki destek e-postasi
3. `privacy/index.html` ve `terms/index.html` icindeki yasal metinler
4. `assets/` altindaki placeholder screenshotlar

## Publish mantigi

Bu klasoru yeni bir GitHub Pages reposunun root'una kopyalarsan direkt calisacak sekilde hazirlandi.
